# Lesson 5 – Array

Mehrere gleichartige Daten werden in Arrays gespeichert.

---

## Aufbau eines Arrays mit Einzel-Werten

Beispiel ist ein Array von Früchten.<br/>
Die eckigen Klammern umrahmen/begrenzen den Array.<br/>
Die einzelnen Werte sind durch ',' getrennt.<br/>
Der Datentyp der Werte ist hier String (Text).<br/>
Die `""` Anführungszeichen" umschliessen jeden einzlenen String-Wert.

```jsx
const fruits = ["Apfel", "Banane", "Kirsche"];
```

Ausgabe als ungeordnete Liste:

```jsx
<ul>
  {fruits.map((fruit, index) => (
    <li key={index}>{fruit}</li>
  ))}
</ul>
```
Erklärung:<br/>
- `map` ist eine Array-Methode, die für jedes Element im Array eine Funktion ausführt und ein neues Array zurückgibt.<br/>
- `key` ist ein spezielles Attribut in React, das hilft, Elemente in einer Liste eindeutig zu identifizieren. Hier wird der Index des Elements im Array verwendet.

---

## Aufbau eines Arrays mit Objekten

Beispiel für ein Array von Objekten ist eine Aufzählung von Früchten mit zusätzlichen Eigenschaften der Früchte.<br/>
Die geschweifte Klammern umrahmen/begrenzen die Objekte.<br/>
Ein Objekt enthält ein oder mehrere key:value pairs, die durch ',' getrennt sind.

Die Eigenschaften haben einen Namen und einen Wert, getrennt durch einen Doppelpunkt.<br/>
Hier sind die Eigenschaften Name, Farbe und Anzahl der Kerne.

```jsx
const fruits = [
   { name: "Apfel", color: "rot", numberOfSeeds: 6 },
   { name: "Banane", color: "gelb", numberOfSeeds: 0 },
   { name: "Kiwi", color: "grün", numberOfSeeds: 200 }
];
```

Ausgabe als Tabelle:

```jsx
<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Farbe</th>
      <th>Kerne</th>
    </tr>
  </thead>
  <tbody>
    {fruits.map((fruit, index) => (
      <tr key={index}>
        <td>{fruit.name}</td>
        <td>{fruit.color}</td>
        <td>{fruit.numberOfSeeds}</td>
      </tr>
    ))}
  </tbody>
</table>
```
