# Lesson 3 – Button-Eventhandler

Häufig sollen Komponenten auf Benutzereingaben reagieren.<br/>
Dazu verwendet man **Eventhandler.**

---

## Aufbau eines Eventhandlers

Ein **Eventhandler ist eine Funktion**, die aufgerufen wird, wenn ein bestimmtes Ereignis eintritt.<br/>
zBsp. Klickt man auf einen Button, so wird in der Funktion wird z.B. ein Zähler hochgezählt oder etwas berechnet.

Für den Zähler braucht es einen UseState:

```jsx
function ClickCounter(){
   const [count, setCount] = React.useState(0);
   ...
```
Funktion zum Hochzählen:

```jsx
function ClickCounter(){
   const [count, setCount] = React.useState(0);

   function handleClick() {
      setCount(count + 1);
   }
   ...
```
Einen Button, der mit dem Eventhandler verbunden ist und<br/>
einen Absatz zur Ausgabe des Zählerwerts:

```jsx
function ClickCounter(){
    ...
    return (
        <>
            <p>Du hast {count} mal geklickt.</p>
            <button onClick={handleClick}>+1</button>
        </>
    );
}
```
